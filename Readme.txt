📄 **LEIA-ME (README)**

**Projeto:** Calculadora de IMC Online – Rápido e Grátis

**Descrição:**
Este projeto consiste em uma calculadora de IMC (Índice de Massa Corporal), desenvolvida para auxiliar usuários a calcular seu índice com base em peso e altura, fornecendo uma classificação de acordo com padrões de saúde.

**Tecnologias utilizadas:**

* PHP
* SQL (banco de dados)
* HTML, CSS e JavaScript

**Objetivo:**
Este projeto foi desenvolvido como trabalho escolar, com o objetivo de aplicar conhecimentos em desenvolvimento web e integração com banco de dados.

**Funcionalidades:**

* Cálculo automático do IMC
* Exibição da classificação (baixo peso, normal, sobrepeso, etc.)
* Interface simples e de fácil uso
